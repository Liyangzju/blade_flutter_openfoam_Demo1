## 1. 项目目标

从 OpenFOAM v2112 官方 `wingMotion` 算例出发，逐步搭建叶片颤振分析原型：

```text
官方翼型动网格算例
-> 自定义模态振动
-> CalculiX FEM 模态导入
-> 三维旋转多叶片 FEM 模态运动
-> IBPA / 气动阻尼 / 颤振判据
```

当前最重要的主线是 Demo3：六叶片旋转叶轮的 PrePoMax/CalculiX 结构模态导入、模态映射、相位控制和 OpenFOAM 动网格计算。

## 3. 归档对话提炼出的路线

第一阶段，找到官方基础：

```text
$FOAM_TUTORIALS/incompressible/pimpleFoam/RAS/wingMotion
```

官方流程是先生成翼型网格，再用 `simpleFoam` 求稳态初场，最后把初场映射到 `pimpleFoam` 动网格算例里做六自由度运动。

第二阶段，建立 Demo1：

```text
二维翼型给定解析模态
pointDisplacement 使用 codedFixedValue
目标是验证自定义模态运动如何驱动 OpenFOAM 动网格
```

第三阶段，建立 Demo2：

```text
demo2A: CalculiX 求壳单元模态
demo2B: OpenFOAM 读取映射后的 FEM 模态
```

核心链路：

```text
CalculiX .inp
-> ccx
-> .dat/.frd
-> surface mode CSV
-> map_fem_mode_to_patch.py
-> OpenFOAM pointDisplacement
```

第四阶段，建立 Demo3：

```text
三维旋转叶片
六个叶片 patch: blade1 ... blade6
轮毂/轴连接固定约束在 PrePoMax 中定义
PrePoMax/CalculiX 使用独立结构网格计算模态
PrePoMax 中命名 blade1...blade6 和 hub surface
每个叶片可通过 IBPA 设置相位差
```

## 4. 当前 Demo3 工作流

当前主算例路径：

```text
demo3/
```

关键文件：

```text
demo3/Allrun

demo3/constant/modeShapes/modeProperties
demo3/0/pointDisplacement
scripts/extract_calculix_frd_modes.py
scripts/map_fem_mode_to_patches.py
```

`demo3/Allrun` 的目标流程：

```text
foamFormatConvert
-> 从 fem/prepomax/Analysis-1.frd 提取模态和频率
-> 选择指定模态或第一条正频率模态
-> 将该模态频率写入 modeProperties
-> 将该模态 CSV 映射回 blade.* patch
-> decomposePar
-> pimpleFoam 并行计算
-> reconstructPar
```

运行方式：

```bash
cd ~/OpenFOAM/liyang-v2112/run/wingMotion/demo3
./Allrun
```

清理方式：

```bash
cd ~/OpenFOAM/liyang-v2112/run/wingMotion/demo3
./Allclean
```

## 5. 当前 Demo3 物理和数值设置

动网格：

```text
dynamicFvMesh   dynamicMotionSolverFvMesh;
motionSolver    solidBodyDisplacementLaplacian;
cellZone        rotter.SLDPRT.rotor;
```

刚体旋转：

```text
origin (0 0 0)
axis   (0 0 1)
omega  158 rad/s
```

模态位移：

```text
patch: blade.*
mode file: bladeMode1_mapped.csv
```

相位控制：

$$
\begin{aligned}
q_b(t)
&=
A\sin\left(2\pi f t+\varphi+b\sigma\right),
\\
b
&=
0,1,\dots,5.
\end{aligned}
$$

其中 \(b\) 是叶片编号，\(\sigma\) 是相邻叶片相位差，也就是 `ibpa`。

| patch | \(b\) |
| --- | ---: |
| `blade1` | 0 |
| `blade2` | 1 |
| `blade3` | 2 |
| `blade4` | 3 |
| `blade5` | 4 |
| `blade6` | 5 |

`ibpa` 位于：

```text
demo3/constant/modeShapes/modeProperties
```

六叶片常用相位示例：

```text
ibpa 0       所有叶片同相
ibpa pi/3    相邻叶片 60 度相位差
ibpa pi      相邻叶片 180 度相位差
```

```text

3. FEM 模态是否需要质量归一化或最大位移归一化。
4. 最近邻映射是否需要升级为 IDW/RBF。
5. 后处理是否计算气动功和气动阻尼。
```
